# 🎮 無限マインスイーパー

ブラウザ上で進捗が永続化され、無限にプレイ可能なマインスイーパーゲームです。

## 📖 概要

**無限マインスイーパー**は、従来のマインスイーパーのコンセプトを拡張し、無限に広がる盤面でプレイできるWebアプリケーションです。ゲームの進捗状況（盤面、得点、ライフ）はローカルストレージに自動保存されるため、いつでも中断した場所から再開できます。

### 主な特徴

- 🔄 **無限に広がる盤面**: ユーザーの操作に応じて動的に生成・拡張
- 💾 **進捗の自動保存**: ローカルストレージによる永続的なデータ保存
- ❤️ **ライフシステム**: 地雷を踏んでも即座にゲームオーバーにはならない
- 🏆 **ハイスコア機能**: 最高得点を記録・表示
- 🎨 **モダンなUI**: Tailwind CSSとShadcn/UIによる洗練されたデザイン

## ✨ 主要機能

### 盤面操作

- **無限に広がる盤面**: ユーザーの操作に応じて盤面を動的に生成・拡張
- **盤面移動**: マウスドラッグによって盤面を自由に移動
- **ズーム機能**: マウスホイールで盤面の表示を拡大/縮小
- **初期状態**: 最初のセル（および隣接する空のセル）が開かれた状態で開始

### ゲームシステム

- **地雷配置**: 設定ファイルで地雷の配置確率を指定可能（初期値: 15%）
- **得点計算**: 開いたセル1つにつき1点加算
- **ライフシステム**:
  - 初期ライフ数: 設定ファイルで指定可能（初期値: 3個）
  - 地雷を踏むとライフが1つ減る
  - 設定した閾値（初期値: 1000ポイントごと）に到達するごとにライフが1つ追加される
- **ゲームオーバー**: ライフが0になったらゲームオーバー。新しいゲームを開始

### データ管理

- **自動保存**: 盤面情報、得点、残りライフをローカルストレージに自動保存
- **保存トリガー**: 地雷を踏んだ時、またはセルの開示時に自動保存
- **ハイスコア**: 過去の最高得点を記録・表示。更新時にリアルタイムで通知

### UI表示

- **主要情報の表示**: 画面上に「現在の得点」と「残りライフ数」を常時表示

## 🛠️ 技術スタック

- **フレームワーク**: React 19
- **ビルドツール**: Vite
- **スタイリング**: Tailwind CSS 4
- **UIコンポーネント**: Shadcn/UI
- **言語**: TypeScript
- **アイコン**: Lucide React

## 📋 動作環境

- **対象環境**: PCブラウザ（モダンブラウザ）

## 🚀 セットアップ

### 必要な環境

- Node.js 24 以上
- npm または yarn

### インストール手順

1. リポジトリをクローン

```bash
git clone <repository-url>
cd mugen_sweeper
```

2. 依存関係をインストール

```bash
npm install
```

3. 開発サーバーを起動

```bash
npm run dev
```

4. ブラウザで `http://localhost:5173` を開く

## 📦 ビルド

本番用のビルドを作成する場合:

```bash
npm run build
```

ビルド結果は `dist` ディレクトリに出力されます。

プレビューを確認する場合:

```bash
npm run preview
```

## ⚙️ 設定

以下の3項目を設定ファイルで変更可能です:

- **地雷の配置確率**: 初期値 15%
- **初期ライフ数**: 初期値 3個
- **ライフ追加の閾値**: 初期値 1000ポイントごと

設定ファイルの詳細については、開発ドキュメントを参照してください。

## 🎮 使い方

1. **セルを開く**: 左クリックでセルを開きます
2. **フラグを立てる**: 右クリック（または設定に応じた操作）でフラグを立てます
3. **盤面を移動**: マウスドラッグで盤面を移動します
4. **ズーム**: マウスホイールで拡大/縮小します
5. **進捗の保存**: 自動的にローカルストレージに保存されます
